// architecture
A cognitive system, modeled on the brain.
Specialized actors that think, remember, and act independently — coordinated through structured message channels.
cortex
Executive layer. Your direct interface. Handles tasks, delegates work, manages context.
hippocampus
Associative memory retrieval. Pattern-completes context and surfaces what's relevant before you ask.
dmn
Default Mode Network. Scans goals in the background, connects dots, catches what you've missed.
amygdala
Emotional salience monitor. Tracks urgency, detects escalation, knows when to interrupt and when to wait.

// principles
Why it works this way.

🔒 Fully local

Runs on your machine. Only LLM API calls leave your computer. Everything else — memory, files, conversations — stays home.

💬 Telegram-native

Message your bot like you'd message a friend. Already installed, already in your pocket, always available.

🧠 Real memory

Working memory, long-term archival, associative recall. A genuine memory system that grows with you over months.

🔓 Open source

MIT licensed. Read every line of code, modify anything, run it forever. It's yours.

// get started
Two minutes to memory.
1

Install

One command. Works on macOS and Linux.

curl -fsSL https://lethe.gg/install | zsh
2

Say hello

Open Telegram and message your bot. That's it. Lethe remembers from here on out.

// you'll need

  • A Telegram Bot Token — message @BotFather, send /newbot
  • An LLM API key — OpenRouter, Anthropic, or OpenAI
  • Your Telegram User ID — message @userinfobot
# full system access (files, commands, everything):
curl -fsSL https://lethe.gg/install | zsh -s -- --unsafe
